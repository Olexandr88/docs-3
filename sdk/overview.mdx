## Installation

<Tabs>
<Tab title="npm">

```bash
npm install --save @chainpatrol/sdk
```

</Tab>

<Tab title="yarn">

```bash
yarn add @chainpatrol/sdk
```

</Tab>

<Tab title="pnpm">

```bash
pnpm add @chainpatrol/sdk
```

</Tab>
</Tabs>
